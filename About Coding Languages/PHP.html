PHP is a general-purpose scripting language geared towards web development. It was originally created by Danish-Canadian programmer Rasmus Lerdorf in 1994. The PHP reference implementation is now produced by The PHP Group.

  1. Implementation language: C (primarily; some components C++)
  2. Stable release: 8.0.9 / 29 July 2021; 10 days ago
  3. Preview release: 8.1.0 Beta 1 / 22 July 2021; 17 days ago
  4. Developer: The PHP Development Team, Zend Technologies
  5. Typing discipline: Dynamic, weak since version 7.0: Gradual
  6. Paradigm: Multi-paradigm: imperative, functional, object-oriented, procedural, reflective
  7. Designed by: Rasmus Lerdorf
