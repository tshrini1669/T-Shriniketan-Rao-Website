C is a general-purpose, procedural computer programming language supporting structured programming, lexical variable scope, and recursion, with a static type system. By design, C provides constructs that map efficiently to typical machine instructions.

  1. Typing discipline: Static, weak, manifest, nominal
  2. Stable release: C17 / June 2018;
  3. Preview release: C2x (N2596) / December 11, 2020;
  4. Typing discipline: Dynamic, weak, duck
  5. Developer: Dennis Ritchie & Bell Labs (creators); ANSI X3J11 (ANSI C); ISO/IEC JTC1/SC22/WG14 (ISO C)
  6. Designed by: Dennis Ritchie
  7. Filename extensions: c,.h

    8. C++

    C++ is a general-purpose programming language created by Bjarne Stroustrup as an extension of the C programming language, or "C with Classes".

    1. Preview release: C++23 / 15 December 2020;
    2. Stable release: C++20 (ISO/IEC 14882:2020) / 15 December 2020;
    3. Typing discipline: Static, nominative, partially inferred
    4. Developer: ISO/IEC JTC1 (Joint Technical Committee 1) / SC22 (Subcommittee 22) / WG21 (Working Group 21)
    5. Designed by: Bjarne Stroustrup
    6. Filename extensions: C,.cc,.cpp,.cxx,.c++,.h,.H,.hh,.hpp,.hxx,.h++
